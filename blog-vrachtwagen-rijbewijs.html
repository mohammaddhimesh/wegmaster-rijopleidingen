Vrachtwagen rijbewijs — rijbewijs C en CE

Wil je als vrachtwagenchauffeur aan de slag? Dan heb je een rijbewijs C (vrachtwagen zonder aanhanger) of rijbewijs CE (vrachtwagen met aanhanger) nodig. In dit artikel lees je alles over de vrachtwagenopleiding, kosten en doorlooptijd.

💡 Wist je dat? Nederland heeft een groot tekort aan vrachtwagenchauffeurs. Met een rijbewijs CE heb je uitstekende kansen op de arbeidsmarkt.

Wat heb je nodig voor rijbewijs C of CE?

  • Minimale leeftijd: 18 jaar (C) of 21 jaar (CE), of jonger via BBL-traject
  • Geldig rijbewijs B
  • Medische keuring
  • Theorie-examen vrachtwagen
  • Praktijkexamen vrachtwagen
  • Code 95 certificaat (verplicht voor beroepsmatig rijden)

Veelgestelde vragen

Hoe lang duurt een vrachtwagen rijbewijs halen?

Gemiddeld 4-8 weken voor rijbewijs C, en 2-4 weken extra voor CE. Dit hangt af van je rijervaring en beschikbaarheid.

Heb ik rijbewijs B nodig voor rijbewijs C?

Ja, rijbewijs B is een verplichte voorwaarde voor het halen van rijbewijs C of CE.

Wat is Code 95?

Code 95 is een vakbekwaamheidscertificaat dat verplicht is voor beroepsmatige vrachtwagenchauffeurs. Je moet dit elke 5 jaar verlengen via 35 uur nascholing.

Kan ik rijbewijs CE halen zonder eerst CE te hebben?

Je kunt direct starten met CE, maar de meeste mensen halen eerst C en dan CE. Bespreek de beste route met ons.

Is er een tekort aan vrachtwagenchauffeurs?

Ja, in heel Europa is er een groot tekort aan gecertificeerde vrachtwagenchauffeurs. Kansen op de arbeidsmarkt zijn uitstekend.
